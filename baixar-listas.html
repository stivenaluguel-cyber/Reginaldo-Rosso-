📥 Atualizar imóveis da Caixa

Baixe as listas oficiais (RS e SC) e suba no GitHub. Leva ~3 minutos.

1

Clique nos dois botões abaixo. Se aparecer uma verificação (CAPTCHA), é só resolver — você é humano, passa fácil. Os arquivos vão para a pasta Downloads.

⬇️ Baixar lista do Rio Grande do Sul ⬇️ Baixar lista de Santa Catarina
2

Suba os 2 arquivos (Lista_imoveis_RS.csv e Lista_imoveis_SC.csv) no seu repositório, substituindo os antigos.

🔗 Abrir o GitHub para subir os arquivos
3

Abra reginaldorosso.com.br/imoveis.html e dê Cmd + Shift + R. Pronto — lista atualizada!

Se o arquivo abrir como texto na tela em vez de baixar: clique com o botão direito na página → "Salvar como…" e salve com o nome Lista_imoveis_RS.csv (ou SC). O importante é o nome ficar exatamente assim.
